Nebenkosten berechnen

Wasser

Strom

Gas

Berechnen Sie ihre Gebühren für das verbrauchte Erdgas. Inklusive anfallende Erdgas- und Umsatzsteuer.


Funktionen

Zählerstände verwalten

Hier können sie ihr Zählerstände verwalten. Diese stellen das Fundament der Nebenkostenberechnung dar. Sie können täglich neue Zählerstände eintragen und so die Nebenkostenberechnung genauer gestalten. Bei einem Eingabefehler können sie Zählerstände auch im Nachhinein noch bearbeiten oder komplett löschen.

Graphische Verbrauchsübersicht

Ein Bild sagt mehr als tausend Worte. Hier können sie ihren Verbrauchsverlauf für alle Nebenkostenarten graphisch betrachten. Der Betrachtungszeitraum ist frei wählbar.

Finanzstatus

Nachdem Sie ihre Zählerstände eines vergangen Zeitraums eingegeben haben, können Sie hier ihre anfallenden Nebenkosten berechnen. Das Programm unterstützt sie automatisch, indem es Ihnen den Verbrauch der ausgewählten Abrechnungsperiode errechnet. Sie müssen nur zusätzliche anfallende Gebühren aus ihren Verträgen ergänzen. Anschließend sehen sie ihre Kosten auf die Periode zugeschnitten.

Graphische Übersicht

Hier können sie ihre Verbrauchsverläufe graphisch betrachten. Wenn Sie einen neuen Zählerstand hinzufügen, bearbeiten oder löschen, wird dies auch sofort im jeweiligen Graphen angezeigt.

Bei jedem Graphen können Sie zudem den betrachteten Zeitraum unterhalb des Graphen einstellen. Soll wieder der gesamte Zeitraum angezeigt, dann kann dies rechts über dem Grapen geändert.
